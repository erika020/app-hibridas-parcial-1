Achievement

POST/achievement — Create achievement

Body (JSON):

{
                "name": "Decorated Agent",
                "description": "Unlock all Trophies in Call Of Duty: Black Ops 6",
                "points": "Platinum"  
            }

Success: 201 with saved document.

GET/achievement — List all achievements

Returns an array of achievement documents.

GET/achievement/:id — Get by id

Return 200 with document or 404 if not found.

PUT/achievement/:id — Update

Return 200 with document or 404 if not found.

DELETE/achievement/:id — Delete

Deletes the achievement and returns 200 on success.

Characters

Routes for character management.

POST/character — Create character

Body (JSON):

{
            "name": "Troy Marshall",
            "description": "Troy Marshall is a main character featured in Call of Duty: Black Ops 6...",
            "rank": ["CIA Agent", "Leader of Rogue Black Ops", "Colonel"],
            "gender": "Male",
            "weapons": ["Model L", "GS45", "DM-10" ...],
            "equipment": ["Flashlight", "Crowbar", "Gas Mask" ...],
            "image": "url"

Return 200 with document or 404 if not found.
